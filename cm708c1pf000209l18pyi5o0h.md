You can install and use **Google Chrome** on **Ubuntu running inside WSL (Windows Subsystem for Linux)** by following these steps:

---

## **Step 1: Enable WSL and Install Ubuntu**

If you haven't already installed WSL and Ubuntu, follow these steps:

1. Open **PowerShell** as Administrator and run:
    
    ```powershell
    wsl --install -d Ubuntu
    ```
    
2. Restart your computer.
    
3. Open **Ubuntu** from the Start Menu and set up your user.
    

---

## **Step 2: Update and Install Dependencies**

Run these commands in the Ubuntu terminal:

```bash
sudo apt update && sudo apt upgrade -y
sudo apt install wget curl gnupg -y
```

---

## **Step 3: Download and Install Google Chrome**

```bash
wget https://dl.google.com/linux/direct/google-chrome-stable_current_amd64.deb
sudo apt install ./google-chrome-stable_current_amd64.deb -y
```

---

## **Step 4: Enable GUI Support (For WSL2)**

If you are using **WSL2**, ensure you have **WSLg (Windows Subsystem for Linux GUI)** enabled:

* **Windows 11 has WSLg by default** (no need to install anything).
    
* If you are using **Windows 10**, update WSL:
    
    ```powershell
    wsl --update
    ```
    

---

## **Step 5: Run Google Chrome**

You can now launch Google Chrome using:

```bash
google-chrome --no-sandbox
```

or

```bash
google-chrome-stable --no-sandbox
```

> **Note:** The `--no-sandbox` flag is required due to WSL's security model.

---

## **Step 6: (Optional) Create a Desktop Shortcut**

If you want to launch Chrome from the Start Menu or a shortcut:

```bash
echo -e '[Desktop Entry]\nName=Google Chrome\nExec=google-chrome-stable --no-sandbox\nIcon=google-chrome\nType=Application\nCategories=Network;WebBrowser;' | sudo tee /usr/share/applications/google-chrome.desktop
```
